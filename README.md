# To install:
```
#linux and macos
git clone https://github.com/0ARTER0/NeoART ~/.config/nvim && nvim

# hope windows install work good
#windows cmd
git clone https://github.com/0ARTER0/NeoART %USERPROFILE%\AppData\Local\nvim && nvim
#windows powershell
git clone https://github.com/0ARTER0/NeoART $ENV:USERPROFILE\AppData\Local\nvim; nvim

```
